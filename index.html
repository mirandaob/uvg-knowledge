GitHub Pages setup — one time only, takes 5 minutes
1
Create a free GitHub account
Go to github.com/signup and create a free account if you don't have one.
2
Create a new repository
Go to github.com/new. Name it something like uvg-knowledge. Set it to Public. Tick Add a README file. Click Create repository.
3
Enable GitHub Pages
In your new repository, go to Settings → Pages. Under Source, select Deploy from a branch. Choose branch main and folder / (root). Click Save. GitHub will give you your permanent URL.
4
Create a personal access token
Go to github.com/settings/tokens/new. Give it a name like uvg-platform. Set expiration to No expiration. Under Scopes, tick repo (the top checkbox). Scroll down and click Generate token. Copy the token immediately — you only see it once.
5
Enter your details below and connect
Fill in the fields below with your GitHub username, repository name, and the token you just created.
